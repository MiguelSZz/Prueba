¿Qué es el diseño web responsive?

El diseño web responsive es una técnica de diseño y desarrollo de sitios web que permite que éstos se adapten automáticamente al tamaño de pantalla del dispositivo en el que se están visualizando.



Esto significa que un sitio web diseñado con técnicas de diseño web responsive se verá bien tanto en un smartphone como en una tablet o en un ordenador de escritorio, sin necesidad de crear diferentes versiones del sitio para cada dispositivo.


El diseño web responsive se logra mediante el uso de CSS y HTML. En CSS se utilizan las reglas de media queries para definir cómo se deben mostrar los elementos de la página en diferentes tamaños de pantalla. Por su parte, en HTML se utilizan las etiquetas semánticas para estructurar el contenido de la página de forma que sea fácilmente legible tanto para los usuarios como para los motores de búsqueda.


Se trata de redimensionar y colocar los elementos de la web de forma que se adapten al ancho de cada dispositivo permitiendo una correcta visualización y una mejor experiencia de usuario. Se caracteriza porque los layouts (contenidos) e imágenes son fluidos y se usa código media-queries de CSS3.
